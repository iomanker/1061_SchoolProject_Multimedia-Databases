# 1061 多媒體資料庫期末專案
## Vue.js連接node.js + mySQL製作SPA型態網路書店
### 前端
* 使用框架: Vue.js v2
* npm版本: 5.5.1
* 指令(於"front-end"下)
```
    npm run dev
```
### 後端
* 使用框架: Express.js + Knex.js + npm-mySQL
* node.js版本: 8.9.1
* 指令(於"back-end"下)
```
    node index.js
```
### mySQL
* 如需更改mySQL連接config，於 back-end/index.js 修改第四行的宣告變數即可
